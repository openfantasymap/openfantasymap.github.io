A maintained fork of the original Blastzone module for FoundryVTT, brought up to v11–v13 and integrated with the shared-world stack. Drop a measured template — circle, cone, ray, or rectangle — and the walls inside it become rubble.

Run it alongside [OFM Shared World](/tools/ofm-shared-world/) and the destruction propagates to every connected table in real time. No more cross-Discord arguments about whether the wall is still there.
