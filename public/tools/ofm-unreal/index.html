OFM Unreal

Generate Unreal Engine scenes from fantasy map styles.

Body

An Unreal Engine plugin that consumes an Open Fantasy Maps style and assembles a scene hierarchy from its layers — at runtime or in-editor. Filters layers, evaluates Mapbox-style expressions, maps map features to actors via Blueprints, and pulls live overlays from geomqtt.

The bridge that puts the atlas inside the engine where the game already runs. The Unity counterpart lives at OFM Unity.
